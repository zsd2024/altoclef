# 任务系统

Alto Clef 的核心是其强大的任务系统，能够自动执行复杂的 Minecraft 操作序列。

## 任务类型

### 资源收集任务

- **挖掘任务**: 自动挖掘指定方块（如钻石、铁矿石）
- **采集任务**: 收集可破坏的方块（如树叶、花朵）
- **击杀任务**: 击败特定生物获取掉落物（如骷髅、苦力怕）
- **农业任务**: 种植和收获农作物（如小麦、胡萝卜）

### 合成任务

- **工作台合成**: 使用 3x3 工作台合成复杂物品
- **背包合成**: 使用 2x2 背包格子合成简单物品
- **熔炉冶炼**: 在熔炉中冶炼物品（如铁锭、玻璃）
- **高炉/烟熏炉**: 使用高级熔炉进行快速冶炼
- **锻造台升级**: 使用下界合金升级装备

### 导航任务

- **路径规划**: 基于 Baritone 的智能路径finding
- **多维度旅行**: 自动在主世界、下界、末地之间传送
- **结构定位**: 自动寻找并前往特定结构（要塞、村庄等）
- **跟随模式**: 跟随指定玩家或实体

### 生存任务

- **自动进食**: 监控饥饿值并自动进食
- **怪物防御**: 主动攻击或躲避敌对生物
- **危险规避**: 避免岩浆、溺水、跌落伤害
- **水桶自救**: 跌落时自动放置水桶

## 任务优先级系统

Alto Clef 使用智能优先级系统来决定任务执行顺序：

1. **紧急任务**: 生存相关（进食、治疗、躲避危险）
2. **用户任务**: 玩家直接命令的任务
3. **自动化任务**: 空闲时的后台任务（如资源收集）

## 任务链

任务以链式结构组织，每个任务链包含多个子任务：

- **UserTaskChain**: 处理用户命令
- **FoodChain**: 处理食物相关任务  
- **MobDefenseChain**: 处理怪物防御
- **MLGBucketFallChain**: 处理跌落保护
- **WorldSurvivalChain**: 处理世界生存逻辑

## 资源目录系统

`TaskCatalogue` 类定义了所有可获取的资源和对应的获取方法：

### 原始资源
- `log`: 获取各种木头
- `dirt`: 获取泥土和草方块
- `cobblestone`: 获取圆石
- `diamond`: 获取钻石

### 合成资源
- `stick`: 合成木棍
- `planks`: 合成木板
- `iron_ingot`: 冶炼铁锭
- `bread`: 合成面包

### 复杂资源
- `wooden_pickaxe`: 合成木镐
- `iron_helmet`: 合成铁头盔
- `ender_chest`: 合成末影箱

## 任务优化

### 批量处理
- 合并相同类型的多个任务以减少重复操作
- 智能路径规划以最小化移动距离

### 缓存系统
- 方块扫描器缓存已探索区域的方块信息
- 物品存储跟踪器缓存箱子内容
- 合成配方跟踪器缓存可用配方

### 条件判断
- 根据当前物品库存决定是否需要收集更多
- 根据环境条件选择最优的获取方法
- 根据维度限制调整任务策略

## 自定义任务

开发者可以通过继承 `ResourceTask` 类创建自定义任务，并在 `TaskCatalogue.init()` 中注册。

> 💡 **提示**: 使用 `@list` 命令可以查看所有支持的资源和任务。
