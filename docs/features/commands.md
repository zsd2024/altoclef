# 命令系统

Alto Clef 提供了丰富的命令系统，允许用户通过聊天命令控制机器人的行为。

## 命令前缀

默认命令前缀为 `@`，可以在配置文件中修改 `commandPrefix` 设置。

## 可用命令

### 帮助和信息

| 命令 | 说明 |
|------|------|
| `@help` | 显示帮助信息 |
| `@status` | 显示当前机器人状态 |
| `@inventory` | 显示背包物品统计 |
| `@coords` | 显示当前位置坐标 |
| `@list` | 列出所有支持的物品和任务 |

### 资源管理

| 命令 | 语法 | 说明 |
|------|------|------|
| `@get` | `@get <物品> [数量]` | 获取指定物品 |
| `@equip` | `@equip <物品>` | 装备指定物品（工具/武器/盔甲） |
| `@deposit` | `@deposit [物品]` | 将物品存入最近的箱子 |
| `@stash` | `@stash` | 将所有非重要物品存入箱子 |

### 导航和移动

| 命令 | 语法 | 说明 |
|------|------|------|
| `@goto` | `@goto <x> <y> <z>` | 移动到指定坐标 |
| `@follow` | `@follow <玩家名>` | 跟随指定玩家 |
| `@locatestructure` | `@locatestructure <结构名>` | 定位并前往特定结构 |

### 系统控制

| 命令 | 说明 |
|------|------|
| `@stop` | 停止当前所有任务 |
| `@pause` | 暂停机器人 |
| `@unpause` | 恢复机器人 |
| `@idle` | 进入空闲模式（生存/防御） |
| `@reloadsettings` | 重新加载配置文件 |

### 高级功能

| 命令 | 说明 |
|------|------|
| `@gamer` | 启动全自动游戏模式（收集、合成、建造、战斗） |
| `@hero` | 英雄模式（专注于击败末影龙） |
| `@food` | 专门收集食物 |
| `@meat` | 专门收集肉类食物 |
| `@test` | 运行测试任务 |
| `@give` | 给予物品（需要权限） |

## 物品名称规范

Alto Clef 支持以下物品标识方式：

### 标准物品名
- 使用 Minecraft 内部物品名称：`diamond`, `iron_ingot`, `crafting_table`
- 支持别名：`lapis` (等同于 `lapis_lazuli`), `pick` (等同于 `pickaxe`)

### 数量指定
- 单个物品：`@get diamond`
- 指定数量：`@get diamond 64`
- 多个物品：`@get iron_ingot 32, gold_ingot 16`

### 特殊目标
- 坐标：`@goto 100 64 200`
- 玩家名：`@follow Steve`
- 结构名：`@locatestructure stronghold`

## 命令组合

可以使用 `&` 连接多个命令：

```bash
@goto 100 64 200 & @get diamond 64 & @goto 0 64 0
```

## 权限和限制

- 所有命令都可以在游戏中直接使用
- 某些命令（如 `@give`）可能需要服务器权限
- 命令执行受当前配置和游戏状态限制

> 💡 **提示**: 使用 `@help <命令名>` 可以查看特定命令的详细用法。
