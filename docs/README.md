# Alto Clef

Alto Clef 是一个 Minecraft Fabric 模组，能够自动执行各种任务，如收集资源、合成物品、建造结构等。它基于 Baritone 路径规划引擎，并扩展了更高级的任务系统。

## 🚀 核心特性

- **自动化资源收集**: 自动挖掘、采集、合成各种 Minecraft 物品
- **智能路径规划**: 基于 Baritone 的高级路径finding系统
- **多维度支持**: 支持主世界、下界和末地的自动导航
- **生存模式优化**: 自动进食、防御怪物、避免危险
- **可扩展任务系统**: 通过命令系统轻松扩展新功能
- **管家模式**: 可以跟随玩家并提供辅助
